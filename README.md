Coding Sourcer

A collection of template scripts and automation workflows for scraping and sourcing across several themes:


- Make report analytics — automation and reporting workflows built with Make (Integromat)
- Meetup extraction — scripts to pull event and member data from Meetup
- Teamtailor scraping — extraction workflows for candidate and job data from Teamtailor
- Hugging Face sourcing — scripts to source and extract profiles/models data from Hugging Face
- GitHub sourcing — scripts to extract developer and repository data from GitHub

Purpose

These templates are meant to be reused and adapted across sourcing projects, providing a starting point for building repeatable extraction and reporting pipelines rather than one-off scripts.
